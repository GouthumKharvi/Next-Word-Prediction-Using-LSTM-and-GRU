# Next Word Prediction using LSTM and GRU

This project is an End-to-End Deep Learning NLP application built using TensorFlow, Keras, and Streamlit for predicting the next word in a sentence using LSTM and GRU Recurrent Neural Networks.

The model is trained on Shakespeare's Hamlet dataset and generates context-based next-word predictions from user input text.

---

# Features

- Next word prediction using Deep Learning
- LSTM and GRU sequence models
- NLP text preprocessing and tokenization
- Sequence padding and embedding layers
- Streamlit interactive web application
- TensorFlow/Keras implementation
- Shakespeare Hamlet dataset training

# Project Workflow

1. Load Shakespeare Hamlet text dataset
2. Perform text preprocessing
3. Tokenize and create input sequences
4. Apply sequence padding
5. Build LSTM and GRU models
6. Train Deep Learning models
7. Predict next word from user input
8. Deploy using Streamlit
